# Debian4Gamers
A Simple Script to custom Debian 12 for best experience for Gaming on Linux



### Installation / update

- **Secure way:**

  - Download install script:

    ```sh
    wget -P /tmp https://github.com/BenyHdezM/Debian4Gamers/raw/main/install.sh
    ```

  - Review it at `/tmp/install.sh`

  - Run it:

    ```sh
    bash /tmp/install.sh
    ```
